# trueautomation-java-example

https://softesis.atlassian.net/browse/TA-888

You must have installed TrueAutomation client, JDK 8 or later.

## How to run test:

* Init project use `trueautomation init` command
 
* Run test

```bash
mvn -Dtest=exampleTest test

```
or

```bash
mvn -Dtest=fb test
```
